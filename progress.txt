## Codebase Patterns
- tRPC client import path from `client/src/lib/trpc.ts` must use `../../../server/routers/index.js` (3 levels up, not 2)
- React 19 requires initial value for `useRef<T>()` — always pass explicit arg like `useRef<T>(undefined)`
- Vitest root resolves from vite config `root: "client/"` — test files go under `client/src/`
- SQLite schema default values matter for backwards compat — use `.default("complete")` on new enum columns
